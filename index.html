About This Game

Welcome to PokeWordle! This website is my project for WDD-130. The goal of the game is to guess the secret Pokémon name within a limited number of guesses, similar to Wordle.

How to Play

  1. Type a Pokémon name and press Enter.
  2. If a letter is in the correct spot, it will be highlighted one color.
  3. If a letter is in the name but in the wrong spot, it will be highlighted a different color.
  4. If a letter is not in the Pokémon’s name, it will stay gray.
  5. Try to guess the Pokémon in 6 tries or less!
